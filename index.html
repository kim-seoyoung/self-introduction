Core Interest

3D Vision

3D Vision은 2D 이미지나 영상으로부터 3차원 공간의 구조, 깊이, 형태를 복원하고 이해하는 컴퓨터 비전 분야입니다. 카메라로 촬영하면 3D 세계의 한 차원(깊이, Z)이 소실되는데, 이 잃어버린 정보를 AI 모델로 되살리는 것이 핵심 과제입니다.

연구 예시

주요 기술

  • · Depth Estimation — 단일 또는 다중 이미지에서 픽셀별 깊이값을 예측하는 기술. Monocular / Stereo 방식으로 나뉨
  • · 3D Reconstruction — 여러 시점의 이미지로 3D 모델을 복원 (SfM, MVS 등의 기하학 기반 접근)
  • · 3D Object Detection — 2D 이미지에서 객체의 3D Bounding Box(위치, 크기, 방향)를 추정. 자율주행 인지의 핵심 기술
  • · NeRF / 3D Gaussian Splatting — 소수의 이미지로 임의 시점의 사실적 렌더링을 생성하는 최신 Neural Rendering 기술
  • · BEV Perception — 자율주행에서 다중 카메라 영상을 Bird's-Eye View로 변환하여 주변 환경을 3D로 인식

적용 분야

  • · 자율주행 — 카메라 기반 주변 환경 3D 인식, 장애물 거리 추정 (Stradvision 업무와 직결)
  • · 로봇 / AR / VR — 공간 매핑, 물체 파지(grasping), 혼합현실 콘텐츠 배치
  • · 의료 / 산업 — CT·MRI 3D 재구성, 디지털 트윈, 건축 측량
Depth Estimation 3D Object Detection NeRF 3DGS BEV SfM / MVS Point Cloud

Core Interest

Spatial VLM

Spatial VLM은 이미지를 보고 자연어로 소통할 수 있는 VLM(Vision-Language Model)에 3D 공간 추론 능력을 결합한 모델입니다. 기존 VLM이 "무엇이 있는가(What)"에 집중했다면, Spatial VLM은 "어디에, 얼마나 떨어져, 어떤 크기로(Where)"까지 이해하고 답할 수 있습니다. 3D Vision의 공간 이해 능력과 LLM의 언어 추론 능력이 만나는 접점에 위치합니다.

기존 VLM과의 차이

  • · 기존 VLM — "사진에 고양이가 있다" (의미 인식 중심, 공간 정보 없음)
  • · Spatial VLM — "고양이는 소파 위, 카메라에서 약 2m 거리에 있다" (물리적 공간 관계까지 추론)

핵심 능력

  • · 거리 추정 — 이미지 속 객체까지의 실제 물리적 거리를 자연어로 답변
  • · 공간 관계 추론 — 객체 간 상대 위치, 크기 비교, 방향 판단
  • · 3D 기반 질의응답 — "이 물건을 집으려면 로봇 팔을 어디로 보내야 해?" 같은 복잡한 공간 질문에 응답

적용 분야

  • · 로봇 조작(Manipulation) — 자연어 명령으로 물체를 정확한 위치에서 파지하고 배치
  • · 자율 내비게이션 — "왼쪽 문으로 3m 이동해" 같은 공간 기반 명령 이해
  • · AR / 혼합현실 — 실제 공간에 가상 객체를 물리적으로 올바른 위치에 배치
  • · 군사 / 감시 — 영상 속 표적의 위치·거리를 자연어로 보고 (한화시스템 업무와 연결 가능)
